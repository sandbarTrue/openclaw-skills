# OpenClaw Skills

Custom skills for OpenClaw agent.

## Skills

### openspec-bg
Run OpenSpec + Claude Code coding tasks in the background (non-interactive mode). Uses GLM-5 by default to save Opus tokens.

### lark-manager
Feishu/Lark document API operations (create/read/edit/permissions). Supports batch writing for long documents.

### feishu-browser
Browser automation for Feishu web operations (send/read messages, create docs, manage permissions).

## Usage
Copy any skill folder to `~/.openclaw/workspace/skills/` and it will be automatically detected.
